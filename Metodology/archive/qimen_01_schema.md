# Схема таблицы t_qumen_chauby (Ци Мэнь Дунь Цзя - Чай Бу)

Таблица предназначена для хранения рассчитанных раскладов Ци Мэнь Дунь Цзя по методу Чай Бу (Chai Bu) для каждого часа из календаря Ба Цзы.

## DDL

```sql
CREATE TABLE IF NOT EXISTS t_qumen_chauby (
    hour_id TEXT NOT NULL,              -- Ссылка на час в t_bazi_hourly
    chart_num INTEGER,                  -- Номер расклада (Ju Number)
    yin_yang TEXT,                      -- Тип Дунь: 'Yin' или 'Yang'
    chart_type TEXT,                    -- Тип карты (пока только 'Часовой')
    palace_no INTEGER,                  -- Номер дворца (1-9) по Ло Шу
    earth_stem TEXT,                    -- Небесный ствол на Земной тарелке
    is_fou_tou_earth INTEGER,           -- Флаг: является ли ствол лидером (Fou Tou) на Земле (1/0)
    heaven_stem TEXT,                   -- Небесный ствол на Небесной тарелке
    is_fou_tou_heaven INTEGER,          -- Флаг: является ли ствол лидером (Fou Tou) на Небе (1/0)
    star TEXT,                          -- Звезда (сокращенный иероглиф)
    is_main_star INTEGER,               -- Флаг: является ли главной звездой (Zhi Fu) (1/0)
    gate TEXT,                          -- Врата (сокращенный иероглиф)
    is_main_gate INTEGER,               -- Флаг: являются ли главными вратами (Zhi Shi) (1/0)
    spirit TEXT,                        -- Дух (сокращенный иероглиф)
    PRIMARY KEY (hour_id, palace_no)
);
```

## Описание полей

- **hour_id**: Уникальный идентификатор часа, соответствующий `t_bazi_hourly.hour_id`.
- **chart_num**: Номер расклада (1-9). Рассчитывается на основе Солнечного сезона и Юаня дня.
- **yin_yang**: Полярность расклада.
    - `Yang`: С Зимнего солнцестояния до Летнего солнцестояния.
    - `Yin`: С Летнего солнцестояния до Зимнего солнцестояния.
- **palace_no**: Номер дворца квадрата Ло Шу (1-9).
    - 1: Кань (Север)
    - 2: Кунь (Юго-Запад)
    - 3: Чжэнь (Восток)
    - 4: Сюнь (Юго-Восток)
    - 5: Центр (обычно пустой или содержит исходные стволы, звезды переносятся)
    - 6: Цянь (Северо-Запад)
    - 7: Дуй (Запад)
    - 8: Гэнь (Северо-Восток)
    - 9: Ли (Юг)
- **earth_stem**: Ствол на земной тарелке (Ди Пань). Расставляется в зависимости от номера расклада и полярности.
- **heaven_stem**: Ствол на небесной тарелке (Тянь Пань). Переносится вместе со звездой.
- **star**: 9 Звезд. Используются сокращенные иероглифы (без префикса 天).
- **gate**: 8 Врат. Используются сокращенные иероглифы (без суффикса 门).
- **spirit**: 8 Духов. Используются сокращенные иероглифы (один знак).
- **is_fou_tou_***: Флаги, указывающие на положение "Главы" (Лидера текущей десятидневки) в раскладе.
- **is_main_***: Флаги, указывающие на Главную Звезду (Чжи Фу) и Главные Врата (Чжи Ши).
