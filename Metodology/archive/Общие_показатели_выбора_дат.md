## 1. 12 Правителей дней

**Название:** 十二建除日 (12 Jian Chu Ri)

**Описание:** Древняя система (существует со II века н.э.), основанная на 12 типах Ци, связанных с определенными днями в определенных месяцах. Позволяет прогнозировать качество дня.

**Методология расчета:**

> **От какого столпа определяется:** МЕСЯЦ (Земная ветвь текущего месяца)

1. День, имеющий ту же земную ветвь, что и текущий месяц, получает индикатор 1 — **Установление**
2. Следующая по порядку земная ветвь получает индикатор 2 — **Устранение**
3. И так далее по кругу

| Индикатор | Название | Описание |
|-----------|----------|----------|
| 1 | **Установление** | Очень сильная энергия. Хорошо для основания чего-то нового, встреч, переговоров, обсуждения свадьбы (но не самой свадьбы), путешествий, переездов, начала обучения. |
| 2 | **Устранение** | День избавления от старого. Хорошо для уборки, ремонта, земляных работ, диеты, развода, увольнения. |
| 3 | **Наполнение** | Увеличивающее влияние. Хорошо для подписания соглашений, открытия сети магазинов, сбора долгов. |
| 4 | **Равновесие** | Благоприятно для тех, у кого слабая позиция. Хорошо для строительства, путешествий, компромиссов. |
| 5 | **Стабильность** | Очень стабильная энергия. Хорошо для брака, долгосрочных проектов, официальных открытий. |
| 6 | **Удержание** | Средний день. Можно использовать для начала новых дел, подписания договоров, но лучше выбрать другой. |
| 7 | **Разрушение** | День разрушения и истощения. Хорошо для лечения хронических болезней, демонтажа. Плохо для свадьбы, новых дел, покупок. |
| 8 | **Опасность** | Средний день. Можно использовать, если есть хорошая звезда. Хорошо для религиозных мероприятий, установления границ. |
| 9 | **Успех** | **Самый благоприятный день из всех 12.** Подходит для любых занятий — свадьба, бизнес, переезд, путешествие, лечение. |
| 10 | **Принятие** | Хорошо для завершения ранее начатого, получения наград, сбора долгов, начала учебы. |
| 11 | **Открытие** | Очень хорошо для официальных открытий, новоселья, вступления в брак, путешествий. |
| 12 | **Закрытие** | Ци неподвижна. Можно только завершать старые дела, строить плотины, устраивать похороны. |

## 2. Три Ша (三煞)

**Название:** 三煞 (San Sha)

**Описание:** Три типа негативной энергии Ци — элемент, находящийся в конфликте с элементом комбинации трех гармоний. Эффект может проявиться в течение года (для годичных Ша) или месяца (для месячных Ша).

**Методология расчета:**

> **От какого столпа определяется:** ГОД и МЕСЯЦ (Земная ветвь)

**Алгоритм:**
1. Найти Земную ветвь года (или месяца)
2. Определить стихию треугольника, в который входит эта ветвь
3. Найти элемент, сталкивающийся с этой стихией
4. Выделить три земные ветви сезона этого элемента — это и есть Три Ша

| Треугольник | Стихия | Столкновение | Три Ша (сезон) | Ветви |
|-------------|--------|--------------|----------------|-------|
| 寅午戌 (Тигр-Лошадь-Собака) | Огонь | Вода | Зима | 亥, 子, 丑 |
| 亥卯未 (Свинья-Кролик-Коза) | Дерево | Металл | Осень | 申, 酉, 戌 |
| 巳酉丑 (Змея-Петух-Бык) | Металл | Огонь | Лето | 巳, 午, 未 |
| 申子辰 (Обезьяна-Крыса-Дракон) | Вода | Земля | Межсезонье | 辰, 丑, 未 |

**Три типа Ша:**

| Ша | Первая ветвь | Вторая ветвь | Третья ветвь | Характер влияния |
|----|-------------|-------------|-------------|-----------------|
| **Ша ограбления** (劫煞) | 亥 | 申 | 巳 | Потери денег, документов, опасность краж |
| **Ша несчастья** (灾煞) | 子 | 酉 | 午 | Телесные повреждения, болезни, ссоры |
| **Годичное Ша** (岁煞/年煞) | 丑 | 戌 | 未 | Задержки, замедления, невыполнение сроков |

> **Месячные Ша опаснее годичных!** Столпы дня и месяца находятся рядом, влияние сильнее.

| Сфера | Ша ограбления | Ша несчастья | Годичное Ша |
|-------|--------------|-------------|-------------|
| A: Личные отношения | ❌ | ❌ | ❌ |
| B: Деловые сделки | ❌ | ❌ | ❌ |
| C: Путешествия | ❌ | ❌ | ❌ |
| D: Финансы | ❌ | ❌ | ❌ |
| E: Карьера/учеба | ❌ | ❌ | ❌ |
| F: Официальные дела | ❌ | ❌ | ❌ |
| G: Медицина | ❌ | ❌ | ❌ |
| H: Другое | ❌ | ❌ | ❌ |

> **Все сферы: ❌ ИСКЛЮЧАТЬ** — Три Ша неблагоприятны для всех важных дел. Можно пренебречь только в краткосрочных делах, не связанных с природой Ша, если выбран хороший час и есть другие позитивные факторы.

## 3. Разрушитель года

**Название:** 岁破 (Sui Po)

**Описание:** Энергия дня направлена против энергии года. День, земная ветвь которого сталкивается с земной ветвью года.

**Методология расчета:**

> **От какого столпа определяется:** ГОД (Земная ветвь года)

**Алгоритм:** Найти земную ветвь, сталкивающуюся с земной ветвью года.

| ЗВ года | Разрушитель | Столкновение |
|---------|-------------|--------------|
| 子 (Крыса) | 午 (Лошадь) | 子午冲 |
| 丑 (Бык) | 未 (Коза) | 丑未冲 |
| 寅 (Тигр) | 申 (Обезьяна) | 寅申冲 |
| 卯 (Кролик) | 酉 (Петух) | 卯酉冲 |
| 辰 (Дракон) | 戌 (Собака) | 辰戌冲 |
| 巳 (Змея) | 亥 (Свинья) | 巳亥冲 |
| 午 (Лошадь) | 子 (Крыса) | 午子冲 |
| 未 (Коза) | 丑 (Бык) | 未丑冲 |
| 申 (Обезьяна) | 寅 (Тигр) | 申寅冲 |
| 酉 (Петух) | 卯 (Кролик) | 酉卯冲 |
| 戌 (Собака) | 辰 (Дракон) | 戌辰冲 |
| 亥 (Свинья) | 巳 (Змея) | 亥巳冲 |

> Смена столпа года обычно происходит 4 февраля (иногда 3 или 5 февраля).

**Эффект:**
- ❌ Неблагоприятно начинать важные дела
- ❌ Плохо для начала долгосрочных дел
- ❌ Плохо для замужества — проблемы в семейной жизни, сложности с беременностью
- ⚠️ Может использоваться для краткосрочных событий, если нет другого выбора и выбран благоприятный час

## 4. Разрушитель месяца

**Название:** 月破 (Yue Po)

**Описание:** Энергия дня, идущая против энергии сезона. Сталкивается с земной ветвью текущего месяца. Оказывает более сильное влияние, чем разрушитель года.

**Методология расчета:**

> **От какого столпа определяется:** МЕСЯЦ (Земная ветвь текущего месяца)

**Алгоритм:** Тот же, что и для разрушителя года, но точка отсчета — земная ветвь месяца.

| ЗВ месяца | Разрушитель |
|-----------|-------------|
| 子 | 午 |
| 丑 | 未 |
| 寅 | 申 |
| 卯 | 酉 |
| 辰 | 戌 |
| 巳 | 亥 |
| 午 | 子 |
| 未 | 丑 |
| 申 | 寅 |
| 酉 | 卯 |
| 戌 | 辰 |
| 亥 | 巳 |

> **В день смены сезона** часть дня может иметь одну энергетику, а другая часть — другую. Необходимо пересчитывать время для своей местности.

**Эффект:**
- ❌ Нельзя начинать никакие важные личные и финансовые дела
- ❌ Не подходит для свадьбы, обещаний
- ❌ Не подходит для хирургических операций (возможны осложнения)
- ❌ Не подходит для открытия бизнеса, подписания договоров
- ❌ Не подходит для поездок и переездов
- ⚠️ Можно завершать старые дела или разрушать старый дом

### Применимость по сферам

| Сфера | Применение |
|-------|-----------|
| A: Личные отношения | ❌ Свадьба, обещания |
| B: Деловые сделки | ❌ Подписание, открытие |
| C: Путешествия | ❌ |
| D: Финансы | ❌ |

## 5. Столкновение Небесных стволов

**Название:** 天干冲 (Tian Gan Chong)

**Описание:** Конфликт между небесными стволами, расположенными рядом (в соседних столпах).

**Методология расчета:**

> **От какого столпа определяется:** Любые соседние столпы (год-месяц, месяц-день, день-час)

**Пары столкновений:**

| Небесный ствол | Столкновение с |
|----------------|----------------|
| 甲 (Цзя) | 庚 (Гэн) |
| 乙 (И) | 辛 (Синь) |
| 丙 (Бин) | 壬 (Жэнь) |
| 丁 (Дин) | 癸 (Гуй) |
| 戊 (У) | ❌ Нет столкновения (центральный элемент) |
| 己 (Цзи) | ❌ Нет столкновения (центральный элемент) |
| 庚 (Гэн) | 甲 (Цзя) |
| 辛 (Синь) | 乙 (И) |
| 壬 (Жэнь) | 丙 (Бин) |
| 癸 (Гуй) | 丁 (Дин) |

**Эффект:** Конфликты, препятствия в начинаниях. При выборе важных дат стремимся к минимуму столкновений — не хотим видеть столкновения года с месяцем, месяца с днем, дня с часом, а также по вертикали (в одном столпе).

### Применимость по сферам

| Сфера | Применение |
|-------|-----------|
| Все сферы | ⚠️ Чем больше столкновений, тем хуже. Для свадьбы, операций, долгосрочных дел → исключать полностью |

## 6. Разъединяющие дни

**Название:** 离日 (Li Ri)

**Описание:** Дни перед четырьмя ключевыми астрономическими точками года. Ци считается устаревшей и бесполезной, в ней нет жизненной энергии. Из-за очень слабой Ци недостает поддержки, добиться успеха сложно.

**Методология расчета:**

> **От какого столпа определяся:** Не зависит от столпов — определяется по астрономическим точкам

**Алгоритм:** Это дни **перед**:
1. Зимним солнцестоянием (冬至) — обычно 21-22 декабря
2. Весенним равноденствием (春分) — обычно 20-21 марта
3. Летним солнцестоянием (夏至) — обычно 21 июня
4. Осенним равноденствием (秋分) — обычно 22-23 сентября

> Нужно учитывать часовой пояс местности! Для Киева (GMT+2) — отнимать 6 часов от пекинского времени.

**Пример для 2010:**
| Событие | Дата в Пекине | Разъединяющий день (Киев) |
|---------|---------------|---------------------------|
| Зимнее солнцестояние | 22 декабря, 7:40 | **21 декабря** |
| Весеннее равноденствие | 21 марта, 1:34 | **19 марта** |
| Летнее солнцестояние | 21 июня, 19:30 | **20 июня** |
| Осеннее равноденствие | 23 сентября, 11:11 | **22 сентября** |

**Эффект:**
- ❌ Неблагоприятно совершать **любые** важные действия
- ❌ Особенно: свадьбу, переезд, изменение позиции кровати
- ⚠️ Возможно расставание или развод (современные Мастера используют намеренно)

## 7. Изнуряющие дни

**Название:** 废日 (Fei Ri)

**Описание:** День означает истощение и угасание. В дне нет энергии. Это последний день времени года.

**Методология расчета:**

> **От какого столпа определяется:** Не зависит от столпов — определяется по календарю

**Алгоритм:** День, предшествующий приходу каждого сезона:

| Сезон | Солнечный сезон | Обычная дата | Изнуряющий день |
|-------|-----------------|--------------|-----------------|
| Весна | 立春 (Приход весны) | 4 февраля | **3 февраля** |
| Лето | 立夏 (Приход лета) | 5 мая | **4 мая** |
| Осень | 立秋 (Приход осени) | 7 августа | **6 августа** |
| Зима | 立冬 (Приход зимы) | 7 ноября | **6 ноября** |

> Даты нужно уточнять по китайскому календарю для конкретного года и часового пояса.

**Эффект:**
- ❌ Неблагоприятно совершать никакие важные дела
- ❌ Свадьба → пара может остаться бездетной
- ❌ Бизнес → убыточный, не будет развиваться

|-------|-----------|

## 13. Влияние Луны

**Название:** 月相 (Yue Xiang)

**Описание:** Фаза Луны оказывает влияние на энергетику дня. Действия, призванные увеличить, приумножить — на растущей Луне. Действия на избавление, завершение — на убывающей Луне.

**Методология расчета:**

> **От какого столпа определяется:** Не зависит от столпов Бацзы — определяется по лунному календарю

| Фаза | Лунные сутки | Характеристика |
|------|-------------|----------------|

| **Убывающая Луна** | 16–30 | Благоприятна для завершения, избавления, освобождения |
| **Критические дни** | 9, 29 | Все идет не так. Можно только завершать ранее начатое. |

**Правила:**
- Открывать бизнес → на **растущей** Луне
- Увольнять сотрудников, расставаться → на **убывающей** Луне
- Начинать диету → на **убывающей** Луне (избавление)

## Алгоритм проверки даты

### Шаг 1: Универсальные исключения
Проверить наличие в дате (если есть → **исключить**):
- [ ] Затмение (солнечное или лунное) или период между затмениями
- [ ] Разъединяющий день
- [ ] Изнуряющий день
- [ ] День потерь / истинных потерь
- [ ] Разрушитель месяца
- [ ] Три Ша (месячные особенно)

### Шаг 2: Общие неблагоприятные факторы
Проверить (при наличии → **избегать**, если есть выбор):
- [ ] Разрушитель года
- [ ] Столкновения Небесных стволов (чем больше, тем хуже)
- [ ] Наказания (особенно для свадьбы, бизнеса, путешествий)
- [ ] Разрушение / вред
- [ ] Одинокие дни (для свадьбы, партнерства)
- [ ] День без возврата (для финансовых операций)
- [ ] Ян Гун Ша

### Шаг 3: Благоприятные факторы
Проверить (при наличии → **приоритет**):
- [ ] 12 Правителей: индикаторы 1, 5, 9 (Установление, Стабильность, Успех)
- [ ] День трех гармоний (особенно для партнерства, свадьбы)
- [ ] Правильная фаза Луны для конкретного действия
- [ ] Хорошие индивидуальные показатели (личный благородный человек, полезный элемент)

### Шаг 4: Специфика события
- [ ] Сопоставить с применимостью по сферам (таблица выше)
- [ ] Проверить специфические ограничения для конкретного события
