# Устранение неполадок (Calk_KMF)

> **Примечание:** Вынесено из `PROJECT.md` для снижения размера основного документа.

---

## Ошибка "Node.js не установлен или отсутствует в PATH"

1. Убедитесь, что Node.js установлен
2. Проверьте, что путь к Node.js добавлен в переменную PATH
3. Перезагрузите компьютер после установки Node.js
4. Запустите `setup_deps.bat` для автоматической настройки

---

## Ошибка "Python не установлен или отсутствует в PATH"

1. Установите Python, отметив галочку "Add Python to PATH"
2. Перезагрузите компьютер
3. Запустите `setup_deps.bat` для автоматической настройки

---

## Ошибка "Порт 8000 уже занят"

1. Убедитесь, что предыдущая копия приложения не запущена
2. Запустите `stop_app.bat` для завершения всех процессов
3. Проверьте, какой процесс использует порт:
   ```
   netstat -ano | findstr :8000
   ```
