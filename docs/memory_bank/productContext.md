# Контекст продукта

## Общее описание
Платформа китайской метафизики с калькуляторами и CRM-системой для работ с различными методиками: Тунгшу, Цимэнь, Бацзы. Проект находится в стабильной эксплуатации (v2.3 по состоянию на 17.02.2026).

## Основные компоненты продукта

### Модули калькуляторов
1. **Бацзы (Китайская астрология)**
   - Расчет столпов (год, месяц, день, час)
   - Система 10 богов
   - Взаимодействия элементов
   - Расширенные методологии:
     * Наинь (Nayin) - элемент и образ
     * Да Гуа (I Ching) - элемент, период, семья, роль
     * Черный Кролик (Joey Yap) - линейный алгоритм с учетом полярности ствола 1-го ЛД
     * Желтый/Черный путь - 12 духов
     * Фазы Ци - 12 фаз для 10 стволов
     * Тун Шу - полная методология (19 таблиц)

2. **Цимэнь Дунцзя**
   - Две системы: Чжи Рэн (Zhi Run) и Чай Бу (Chai Bu)
   - Годовые, месячные, дневные, часовые расклады
   - Методология Ri Jia для дневных раскладов
   - Оптимизированные шаблоны (1080 уникальных карт)

3. **Тунгшу (Китайский календарь)**
   - Солнечные термины
   - Выбор дат (СКДГ)
   - Черный кролик (исправленный алгоритм версии Joey Yap)

4. **Фэн-шуй**
   - Летящие звезды для периодов: Год, Месяц, День, Час
   - Корректный расчет относительно Ли Чунь
   - 9 дворцов с соответствующими звездами

5. **Rule Engine для анализа**
   - 100+ правил для анализа Ци Мэнь структур
   - Гибридная архитектура (SQL + Python)
   - Области: Дата, Направление, Активация

## Данные
- **Период:** 2025-2027 гг.
- **Часовые пояса:** UTC-12 ... UTC+14
- **Записей:** ~350K в t_bazi_hourly, ~2M в t_qumen_dgiren_hourly

## Целевые функции
- Точные калькуляторы для различных систем
- Интеграция результатов в CRM-систему
- Масштабируемая архитектура для будущего расширения функциональности
