# Функциональные возможности (Calk_KMF)

> **Примечание:** Вынесено из `PROJECT.md` для снижения размера основного документа.

---

## Календарь Тун Шу

- Просмотр календаря на год / месяц / неделю / день
- Детальная информация по каждому часу выбранного дня (12 двухчасовых слотов: столп, 10 богов, фаза Ци, скрытые стволы, шэнь ша)
- Информация о благоприятных и неблагоприятных днях
- Чёрный Кролик (Joey Yap / Wu Tu Ze Ri Fa) — ежедневная звезда с оценкой
- Лунный день и лунный месяц (китайский календарь)
- Компактное отображение Летящих Звёзд (Г/М/Д summary + 3×3 сетка)

---

## Расклады Ци Мэнь

- Выбор различных методологий расчета (Чжи Жунь, Чай Бу)
- Просмотр текущего расклада
- Расчет расклада на выбранную дату
- Детальная информация по дворцам

---

## CRM-система

- Управление базой клиентов
- Ведение истории сессий
- Связывание расчетов с клиентами
- Добавление заметок
