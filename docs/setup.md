# Установка и запуск (Calk_KMF)

> **Примечание:** Инструкции вынесены из `PROJECT.md` для снижения размера основного документа.
> Обновлять при изменении процесса установки.

---

## Системные требования

- Windows 10 или новее
- Python 3.8 или выше
- Node.js 14 или выше
- npm 6 или выше
- 4GB RAM (рекомендуется)
- 500MB свободного дискового пространства

---

## Автоматическая установка

1. Запустите `setup_deps.bat` для автоматической установки и настройки всех зависимостей
2. Следуйте инструкциям на экране
3. После успешной установки всех зависимостей, скрипт завершит работу

---

## Ручная установка

### Python и необходимые библиотеки

1. Установите Python с [официального сайта](https://www.python.org/downloads/)
2. Убедитесь, что отмечена опция "Add Python to PATH" при установке
3. Установите необходимые библиотеки:
   ```
   pip install fastapi uvicorn sqlalchemy psycopg2-binary python-dotenv
   ```

### Node.js и npm

1. Установите Node.js с [официального сайта](https://nodejs.org/)
2. Перейдите в директорию frontend и установите зависимости:
   ```
   cd frontend
   npm install
   ```

---

## Запуск и использование

1. Запустите `start_app.bat` для одновременного запуска бэкенда и фронтенда
2. Скрипт проверит наличие всех зависимостей и предложит установить отсутствующие
3. После успешного запуска, приложение будет доступно по адресам:
   - Фронтенд: http://localhost:3000
   - Бэкенд API: http://localhost:8000
   - Документация API: http://localhost:8000/docs

---

## Остановка

1. Запустите `stop_app.bat` для корректного завершения работы приложения
2. Скрипт найдет и завершит все процессы, связанные с приложением
